Service Table

Service Description
Hadoop Hadoop is a framework that allows for the distributed processing of large data sets across clusters of computers using simple programming models.
HDFS It is a distributed, fault-tolerant file system designed to store and manage large volumes of data across a distributed cluster. It is a core component of Hadoop, providing the scalable storage layer needed to store data for big data processing frameworks like MapReduce, Spark, Hive, and others. HDFS is optimized for high throughput, scalability, and reliability, making it suitable for processing and storing big data.
HIVE It is a data warehouse system built on top of Hadoop that enables the management, summarization, querying, and analysis of large datasets stored in Hadoop’s HDFS (Hadoop Distributed File System). It provides a high-level interface for interacting with data, which allows users to run SQL-like queries on data stored in Hadoop, without needing to write complex MapReduce code.
YARN (Yet Another Resource Negotiator) It is the resource and job management layer in Hadoop that enables efficient utilization of cluster resources and supports diverse applications and workloads
MapReduce is a programming model and processing technique used for handling large-scale data processing in a distributed computing environment, most commonly associated with the Hadoop ecosystem. It enables the parallel processing of massive datasets across multiple machines, breaking down complex tasks into smaller, manageable pieces
Zookeeper Zookeeper is a centralized service for maintaining configuration information, naming, and providing synchronization across distributed systems. It is used by many Hadoop ecosystem components to manage configuration and coordination tasks.
HBase HBase is a distributed, scalable, NoSQL database that is modeled after Google's Bigtable. It runs on top of HDFS and is used for real-time read/write access to large datasets. start-hbase.sh is used to start the HBase service.
Hive Hive is a data warehouse system built on top of Hadoop for providing data summarization, query, and analysis. It allows querying of data using a SQL-like language called HiveQL. The command hive starts the Hive shell, and beeline is a command-line interface for HiveServer2.
Phoenix Phoenix is a SQL skin for HBase, providing an interface to interact with HBase using SQL queries. It runs as a client application and connects to HBase via Zookeeper. sqlline.py is used to launch the Phoenix shell to execute SQL commands.
Zeppelin Apache Zeppelin is a web-based notebook that enables interactive data analytics and collaborative data science. It supports multiple languages, including Scala, Python, and SQL, and can integrate with Hadoop and other big data tools. zeppelin-daemon.sh start is used to start the Zeppelin service.
Apache2 Apache2 is an open-source web server software that provides HTTP services for serving web content. It is commonly used in the Hadoop ecosystem for serving web interfaces. sudo service apache2 start starts the Apache web server.
